# Bellabeat Case Study 

## 📊 Project Overview 
This project examines fitness data from smart devices to uncover patterns in user activity and deliver data-driven recommendations for Bellabeat.

## 🎯 Business Task 
This project analyzes fitness tracker data to generate insights that inform Bellabeat’s marketing strategy and enhance user engagement.

## 🛠️ Tools Used
- R
- SQL
- Tableau
- Google Slides

## 🗂️ Datset 
Fitbit Fitness Tracker Data (Kaggle) 

## 🔍 Process 
- Cleaned and prepared the data using R.
- Analyzed patterns in steps, physical activity, and calorie expenditure.
- Used SQL for additional data exploration.
- Created visualization in Tableau
- Presented findings with supporting visuals

## 📈 Key Insights 
- Higher step counts are associated with increased calorie expenditure.
- Users spend a significant portion of their time in sedentary activity.
- Activity levels fluctuate throughout the week.

## 💡 Recommendations 
- Encourage users to meet their daily step goals.
- Reduce sedentary behavior by incorporating regular activity reminders.
- Promote consistent daily activity habits.
